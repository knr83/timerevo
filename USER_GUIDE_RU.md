# Timerevo — Руководство пользователя

## Установка Timerevo

1. Распакуйте архив timerevo-*.zip в удобную папку
   (например, C:\Program Files\timerevo или C:\Users\<ваше_имя>\timerevo)

2. Запустите timerevo.exe

3. При первом запуске программа создаст базу данных в папке данных приложения
   (обычно: %LOCALAPPDATA%\timerevo\ или аналогичная)

Требования: Windows 10/11 (64-bit)

## Запуск
Дважды щёлкните по `timerevo.exe`.

---

## Терминал (главный экран)

1. Выберите сотрудника из списка.
2. Введите PIN, если требуется (для сотрудников с включённым PIN).
3. При первом входе — **подтверждение политики** (Privacy Policy и Terms; поставьте галочку для продолжения).
4. Нажмите **ПРИШЁЛ** для начала смены или **УШЁЛ** для её окончания.
5. **Мой рабочий календарь** — календарь смен и отсутствий; из календаря можно подать заявку на отсутствие.
6. **Отчёт времени (PDF)** — экспорт PDF по периоду (Сегодня / Неделя / Месяц / Своя дата).
7. Список сегодняшних смен; «Показать ещё N» — развернуть все.
8. Если смена не закрыта после конца рабочего дня — появится блок закрытия с выбором даты и времени.
9. Кнопка **Администрирование** (в шапке приложения) открывает панель администратора.

## Администрирование

Введите PIN администратора для входа. Для выхода нажмите **Назад** — админка блокируется автоматически.

> При первом входе используйте PIN по умолчанию: `0000`.
> Для безопасности смените его после входа в **Настройки → Смена PIN**.

Меню **Помощь**: Политика конфиденциальности, Условия использования, О приложении.

### Сотрудники
- **Добавить** (➕): создать нового сотрудника.
- Выберите сотрудника для просмотра или редактирования карточки.
- Обязательно: код, имя, фамилия.
- По желанию: дата найма, роль, тип занятости, часы в неделю, отдел, должность, контакты.
- **Включить PIN**: требовать PIN при отметке; установить или сбросить PIN для каждого сотрудника.
- **Включить NFC**: альтернативный вход по NFC-токену.
- Назначьте **шаблон расписания**.
- Отметьте активен/неактивен — показывать или скрывать в терминале.

### Расписания
- Создание и управление шаблонами расписаний (например, «Стандартная неделя»).
- Добавление шаблонов — кнопка ➕; меню ⋮ — переименование, активация/деактивация.
- Настройка рабочих интервалов по дням недели (Пн–Вс).
- Укажите время начала и окончания; переключатель «ночная смена» для смен через полночь.
- Назначайте шаблоны сотрудникам в карточке сотрудника.

### Журнал
- Просмотр рабочих смен в таблице (Сотрудник, Начало, Конец, Длительность, Статус).
- Фильтры: период (С/По), пресеты (Сегодня, Неделя, Месяц, Прошлый месяц), **Статус** (Все / Открытые / Закрытые), сотрудник, **Поиск** (по имени или заметке).
- Чипы дат для установки или сброса фильтров.
- Редактирование строки: начало/конец, заметка, причина изменения (обязательно), «Установить конец сейчас», «Сбросить конец».

### Отсутствия
- Фильтры: сотрудник, статус (Все / Ожидает / Одобрено / Отклонено), период (С/По).
- Таблица: Сотрудник, Тип, С/По, Статус, Кто одобрил, Когда.
- Типы: отпуск, больничный, неоплачиваемый отпуск, декрет, учебный отпуск, другое.
- **Добавить отсутствие** — кнопка для создания записи.
- По ожидающим: Редактировать, Удалить, Одобрить, Отклонить (при отклонении — обязательная причина).

### Отчёты
- Просмотр сводки по сотрудникам и периодам. В итоги попадают только **закрытые** смены.
- Фильтр по датам (чипы С/По).
- Экспорт PDF: общий или по выбранному сотруднику из детальной панели.

### Настройки
- **Язык**: системный, немецкий, русский или английский.
- **Тема**: системная, светлая, тёмная, высокая контрастность (светлая), высокая контрастность (тёмная).
- **Рабочее время**: начало и конец для терминала (ограничения отметок).
- **Смена PIN** — изменить PIN администратора.
- **Экспорт диагностики** — сохранение диагностических данных в файл (для техподдержки).
- **«Создать резервную копию»** — сохраняет базу (timerevo.sqlite) в выбранную папку.
- **«Восстановить из резервной копии»** — выбор .sqlite-файла, подтверждение.

## Данные (для администраторов)

**Путь к базе данных:**
- Папка: `%LOCALAPPDATA%\timerevo\`
- Файл: `timerevo.sqlite`

**Полное удаление данных:**
1. Закройте приложение.
2. Удалите папку `%LOCALAPPDATA%\timerevo\` (например: `C:\Users\<пользователь>\AppData\Local\timerevo`).
